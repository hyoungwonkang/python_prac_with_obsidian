## 발견·결정

### 핵심 발견
- **PyTorch는 2.3부터 macOS x86_64 휠을 끊었음**. PyPI에서 직접 확인 — 2.3~2.9 어느 버전도 인텔 맥 휠 없음. 소스 빌드는 가성비 0.
- 교재가 GPU 없는 학습자용으로 **Colab을 권장**한 게 사용자 상황과 정확히 일치 → 교재 안내 그대로 따르는 게 정답.
- **`torch.load(weights_only=True)`가 2.6 기본값**으로 변경된 것이 2.2↔2.6 사이 유일한 의미있는 breaking change. 양쪽 호환을 위해 명시 표준화.

### 결정
- **로컬 = 텐서 문법·소규모 실험·코드 작성 + 디버그**. **Colab = 교재 재현·풀 학습**. 이 분담을 [[../30-References/pytorch-env-hybrid]]에 표로 박음.
- 디바이스는 **항상 `cuda → mps → cpu` 자동선택**. `device='cuda'` 하드코딩 금지.
- 노트북은 GitHub vault가 정본. Colab은 "파일 → GitHub에 사본 저장"으로 직접 푸시 → 로컬에서 merge.
- 의미있는 결정은 묻지 않고 자동 커밋, push는 사용자가 "푸시"라 말할 때만 — 레포 규칙 그대로 적용.
