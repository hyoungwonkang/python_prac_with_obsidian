## 실행 방법

### 로컬 (인텔 맥, PyTorch 2.2.2)
```bash
source ~/ml-env/bin/activate
cd ~/dev/python_prac_with_obsidian/10-Projects/pytorch-study
python ch01_basics.py
```

### Colab (PyTorch 2.6.0, T4)
1. [colab.research.google.com](https://colab.research.google.com) → 새 노트북
2. 런타임 → 런타임 유형 변경 → **T4 GPU**
3. `ch01_basics.py` 내용을 셀에 붙여넣기 또는 GitHub에서 직접 열기
4. Colab은 PyTorch 기본 설치되어 있음 (`pip install` 불필요)
