# Python Practice Vault

파이썬 학습과 프로젝트 작업을 위한 Obsidian vault.

## 구조

| 폴더 | 용도 |
|---|---|
| `00-Inbox/` | 빠른 캡처 (분류 전 메모, 정리 후 다른 폴더로 이동) |
| `10-Projects/` | 진행 중인 프로젝트 단위 계획·할일 |
| `20-Snippets/` | 재사용 가능한 코드 조각 |
| `30-References/` | 외부 문서, 링크, 학습 자료 |
| `90-Daily/` | 데일리 노트 (Daily Notes 플러그인) |
| `99-Archive/` | 보류·중단된 작업 보관 |

## 허브 노트

- [[10-Projects/_Projects]] — 모든 프로젝트 인덱스
- [[20-Snippets/_Snippets]] — 스니펫 인덱스
- [[30-References/_References]] — 레퍼런스 인덱스

## 첫 사용 시 Obsidian 설정

1. **Settings → Files & Links**
   - Default location for new notes: `In folder specified below` → `00-Inbox`
   - New link format: `Relative path to file`
2. **Core plugins**: Daily notes, Templates, Backlinks, Tags view, Quick switcher
3. **Daily notes** 폴더: `90-Daily`
