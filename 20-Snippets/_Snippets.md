# Snippets 인덱스

재사용할 만한 코드 조각 모음. 노트 명명 규칙: `언어-주제.md` (예: `python-regex-validation.md`).

## 태그 기반 분류

- `#python` — 파이썬 일반
- `#regex` — 정규식
- `#cli` — 커맨드라인 도구
- `#io` — 파일/JSON 입출력
- `#validation` — 입력 검증

> Obsidian의 **Tags view** (좌측 사이드바)로 태그별 노트 모아보기 가능.

## 새 스니펫 추가 방법

1. `20-Snippets/` 폴더에 노트 생성: `python-주제.md`
2. 첫 줄에 태그 라인: `tags:: #python #regex`
3. 코드는 ` ```python ` 코드블록으로 감싸기
4. 짧은 사용 예시·주의점 함께 기록
