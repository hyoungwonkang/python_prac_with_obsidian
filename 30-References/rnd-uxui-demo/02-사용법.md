# 02 사용법 — 탐지 AI 통합 데모 (2026-07-10)

## 실행

```bash
cd 30-References/rnd-uxui-demo/export
~/rnd-env/bin/python app.py
# → 브라우저에서 http://127.0.0.1:7860
```

- 사전 조건: `~/rnd-env` — 지시 1 환경 requirements 설치 후 이 폴더의 `requirements.txt`(gradio) 추가 설치.
  NER 가중치 `rnd-detection-models/export/ner_klue.pt` 존재 (없으면 finetune_ner.py로 재생성).
- 모델은 **탭별 첫 분석 때 로딩** — 첫 응답만 수 초~수십 초, 이후 즉시.

## 텍스트 탭

문장 입력 → 분석 버튼. 세 칸이 한 번에 나온다:

1. **스팸 분류 4종 판정** — BERT / RULE / HYBRID-AND / HYBRID-OR (지시 2의 선택지 메뉴).
   판정이 갈리는 문장을 넣어보면 방법 간 성격 차이가 보임.
2. **PII 마스킹** — 마스킹 문장 + 탐지 표(유형·**민감도**·확신) + **종합 위험도와 판정 근거**.
   - 민감도 = 유형의 민감 등급(항목 속성) / 종합 위험도 = 이 문장으로 개인이 특정되는가(문장 판정 — 식별자·준식별자 조합 논리)
   - 토큰 자리의 "확신 애매 → 사람 검토" = 탐지는 됐지만 자동 마스킹을 보류한 항목(REVIEW)
3. **NER 개체** — 인명·지명·기관·날짜·시간·수량 추출.

시연용 예시 문장:
```
신청인 홍길동 (880101-1234568) 무료 쿠폰 당첨! 010-1234-5678로 연락주세요
```

## 검수·라벨링 탭 (수도 라벨링 — 데이터 자산 만들기)

문장 입력 → **초안 생성** (BERT·RULE의 제안과 확신 표시, 불일치면 "검수 가치 최고" 경고)
→ 사람이 판단해 **[정상으로 확정] / [스팸으로 확정]** 클릭 → 저장.

- 저장 위치: `rnd-dataset-artifacts/export/datasets-own/ko-spam-reviewed.csv` — **git 추적 자산** (재생성 불가한 사람의 라벨)
- 누적분 활용 (⚠️ 조건 있음): **정상·스팸 양쪽이 수십 건 이상** 쌓여야 단독 학습 가능
  (소량이면 70:10:20 분할 후 train이 0건 — 학습기가 안내 메시지로 멈춤).
  정석은 순환도대로 **기존 train에 편입**: 검수 CSV의 text,label을 기존 ko/train.csv에 합쳐 학습
  (고정 val·test는 그대로 유지 — 잣대 불변)
- 원칙: 모델 초안은 참고일 뿐 — **확정 버튼을 누르는 순간에만 정답이 됩니다.** 초안이 틀렸다고
  판단되면 반대 버튼을 누르세요 ("초안을 뒤집은 건수"로 집계됩니다)

## 이미지 탭

이미지 업로드 → 분석 버튼:

1. **YOLO 박스** — 무엇이 어디에 (COCO 80클래스, 신뢰도 0.25 기준).
2. **KoCLIP 상황 태그** — 전체 상황의 확신 순위.
   **상황 후보 편집창(한 줄 = 후보 1개)을 고치고 다시 분석하면 재학습 없이 분류 체계가 바뀐다** — 데모의 핵심 포인트.
   1위 확신이 50% 미만이면 "후보에 없는 상황" 경고 표시.

프롬프트 수칙 (지시 4 실측): 부정형 금지 · 대비쌍 · 캐치올 클래스("일상 생활 사진") 유지 · 한국어로.

## 이미지 검색·일괄 탭 (다중 이미지 × 다중 문장)

이미지 여러 장 업로드 + 상황 후보 + **검색 문장** → 일괄 분석. 유사도 행렬 [이미지 N × 문장 M]을
한 번 계산해 **두 방향으로** 읽는다:

- **열 방향 = 검색**: 검색 문장과 가까운 순으로 정렬 (표의 유사도·갤러리) — "이미지 더미에서
  도박 의심 장면 추리기". 순위만 신뢰하고 상위부터 사람이 검토 (검색에는 억지 배정 문제 없음 — 순위일 뿐).
- **행 방향 = 일괄 판정**: 이미지마다 1위 상황과 확신 (기존 이미지 탭의 판정을 N장에 일괄 적용).

실측 예: 혼합 10장에 검색 문장 "스포츠 경기 사진" → 야구 2장이 1·2위(코사인 0.266/0.237).

## 포트·문제 해결

- 7860 포트 사용 중이면 `app.py` 마지막 줄의 `server_port` 변경.
- MPS 관련 오류 시: 환경변수 `PYTORCH_ENABLE_MPS_FALLBACK=1` (app.py가 기본 설정함).
