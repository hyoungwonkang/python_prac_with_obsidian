# 4. 가이드 — 함정·재현 (배치2 OpenCV 부)

작성일: 2026-07-08

## 실전 함정

### ① cv2 5.0 빌드에 objdetect(Haar) 없음 ★ 이번 최대 함정
- `cv2.CascadeClassifier` / `cv2.objdetect` 가 **존재하지 않음** (`opencv-python==5.0.0` 기준). `cv2.data.haarcascades` 폴더도 비어 있음.
- 증상: `AttributeError: module 'cv2' has no attribute 'CascadeClassifier'`.
- **대응**: 얼굴/사람 검출을 OpenCV 단독으로 하지 말 것. **검출은 YOLO, 비식별 처리만 OpenCV**로 분담(본 R&D 방식). 굳이 Haar가 필요하면 XML을 별도 배포하고 `cv2.CascadeClassifier(xml_path)`로 로드하되, 5.0 빌드에 API 자체가 없으면 `opencv-contrib-python` 또는 DNN 검출기로 우회.

### ② imread가 None을 반환
- 경로 오타·한글 경로·손상 파일·HTML(다운로드 실패분)이면 `imread`가 예외 없이 **None**을 준다.
- 대응: 로드 직후 `if img is None:` 확인. 본 스크립트는 SOURCE 실패 시 샘플/합성으로 폴백.

### ③ GaussianBlur 커널은 홀수·양수
- 커널 크기가 짝수/0이면 에러. `deidentify`는 `(min//2)|1`로 홀수 보장, 최소 15.

### ④ VideoWriter 코덱
- macOS에서 `mp4v` fourcc는 동작 확인. 코덱 미지원 환경이면 파일이 0바이트로 생성될 수 있음 → 저장 후 `VideoCapture`로 되읽어 프레임 수 확인(본 스크립트가 수행).

### ⑤ 색 순서 BGR (RGB 아님)
- OpenCV는 **BGR** 기본. RGB 기대 라이브러리(matplotlib·PIL)로 넘길 때 `cv2.cvtColor(..., COLOR_BGR2RGB)` 필요.

### ⑥ YOLO 최초 실행 = 인터넷
- `YOLO("yolov8n.pt")`는 최초 1회 가중치를 캐시로 받는다(오프라인이면 미리 배포). 배치1의 `yolov8n.pt`를 재사용해도 됨.

## 재현 절차 (스모크)

```bash
~/rnd-env/bin/python opencv_basics.py       # 엣지 픽셀 수 출력되면 OK
~/rnd-env/bin/python opencv_frames.py        # "저장: 12장" 류 출력
~/rnd-env/bin/python opencv_deidentify.py    # "검출된 'person': N개"
~/rnd-env/bin/python opencv_pipeline.py      # 프레임별 검출 수
```

## 검증 기준
- 4개 스크립트가 모두 산출물(`opencv_out/*`)을 남기고 에러 없이 종료.
- 비식별: 선명도(라플라시안 분산)가 원본 대비 뚜렷이 감소(실측 100.7→13.8).
