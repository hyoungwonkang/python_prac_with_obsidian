# 3. 사용법 — 배치2 (OpenCV 부)

작성일: 2026-07-08

## 0. 환경

배치1과 같은 `~/rnd-env`(Python 3.12) 공유. OpenCV만 추가:

```bash
# 이미 rnd-env가 있으면 OpenCV만
~/rnd-env/bin/pip install opencv-python==5.0.0

# 처음이면
pip install -r requirements.txt
```

검증:
```bash
~/rnd-env/bin/python -c "import cv2; print(cv2.__version__)"   # 5.0.0
```

## 1. 기본 처리

```bash
cd <이 폴더>
~/rnd-env/bin/python opencv_basics.py
# 다른 이미지로:
SOURCE=/path/to/img.jpg ~/rnd-env/bin/python opencv_basics.py
```
→ `opencv_out/basics/`에 gray·resized·blur·edges PNG.

## 2. 영상 프레임 추출

```bash
~/rnd-env/bin/python opencv_frames.py                       # 합성 영상 자동 생성·분해
VIDEO=/path/clip.mp4 EVERY=10 ~/rnd-env/bin/python opencv_frames.py
```
→ `opencv_out/frames/frame_XXXX.png`.

## 3. 비식별화 (YOLO 검출 + OpenCV 블러)

```bash
~/rnd-env/bin/python opencv_deidentify.py                   # zidane.jpg, 블러
SOURCE=/path/img.jpg MODE=mosaic ~/rnd-env/bin/python opencv_deidentify.py
TARGET=car ~/rnd-env/bin/python opencv_deidentify.py        # 다른 COCO 클래스
```
→ `opencv_out/deid/deidentified.png`. 최초 1회 `yolov8n.pt` 자동 캐시(인터넷).

## 4. end-to-end 파이프라인

```bash
~/rnd-env/bin/python opencv_pipeline.py                     # 샘플 2장
VIDEO=/path/clip.mp4 EVERY=15 ~/rnd-env/bin/python opencv_pipeline.py
```
→ `opencv_out/pipeline/deid_XXXX.png` + 프레임별 검출 수.

## 5. 결과 확인 (선택)

비식별이 실제로 정보를 지웠는지 수치 확인:
```bash
~/rnd-env/bin/python - <<'PY'
import cv2
def sharp(p):
    im = cv2.imread(p); g = cv2.cvtColor(im, cv2.COLOR_BGR2GRAY)
    return cv2.Laplacian(g, cv2.CV_64F).var()
print("비식별 선명도:", sharp("opencv_out/deid/deidentified.png"))  # 낮을수록 흐림
PY
```
