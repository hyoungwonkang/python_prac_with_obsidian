## A. OpenCV 스크립트 4종

### `opencv_basics.py` — 기본 처리
- `load_image()`: `SOURCE` → ultralytics 샘플(bus.jpg) → 합성 이미지 순으로 확보.
- 처리 4종: `cvtColor(BGR2GRAY)` / `resize`(가로 640, 비율 유지, INTER_AREA) / `GaussianBlur(25×25)` / `Canny(100,200)`.
- 산출: `opencv_out/basics/{gray,resized,blur,edges}.png`.

### `opencv_frames.py` — 영상 프레임 추출
- `make_synth_video()`: `VideoWriter`(mp4v)로 움직이는 흰 원 60프레임 합성 — 외부 영상 없이 검증.
- 본체: `VideoCapture`로 열어 `EVERY`프레임마다 `imwrite`.
- 파라미터: `VIDEO`(입력 경로) · `EVERY`(기본 5).

### `opencv_deidentify.py` — 비식별화 (핵심)
- `find_source()`: `SOURCE` 또는 ultralytics zidane.jpg.
- **`blur_roi(img, x1,y1,x2,y2, mode)`**: 박스 영역만 처리해 되붙임.
  - `blur`: 영역 크기에 비례한 홀수 커널 `GaussianBlur`.
  - `mosaic`: 1/16 축소 후 `INTER_NEAREST` 확대 = 모자이크.
- 본체: `YOLO("yolov8n.pt")`로 검출 → `TARGET`(기본 person) 박스마다 `blur_roi`.
- **역할 분담**: 검출=YOLO(사전학습 COCO 80클래스), 처리=OpenCV. (cv2 5.0 objdetect 부재 대응)

### `opencv_pipeline.py` — end-to-end
- `iter_frames()`: `VIDEO` 있으면 분해, 없으면 샘플 2장을 '프레임 묶음'으로.
- 각 프레임: YOLO 검출 → `blur_roi`(재사용, `from opencv_deidentify import blur_roi`) → 저장.
- 파라미터: `VIDEO` · `EVERY`(15) · `TARGET`(person) · `MODE`(blur|mosaic).

## 공통 규약 (배치1과 동일)
- `HERE = Path(__file__).resolve().parent`, 산출은 `opencv_out/` 하위.
- YOLO 사용 스크립트는 `PYTORCH_ENABLE_MPS_FALLBACK=1`, device = mps→cuda→cpu 자동.
- 환경변수로 파라미터화(`SOURCE`·`VIDEO`·`EVERY`·`TARGET`·`MODE`).
