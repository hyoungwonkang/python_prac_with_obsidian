# 04 가이드 (작업본) — "분류 잘하는 법" 판단 틀

> 보고용 자립본: [[export/4_가이드]]

## 요지 — 새 분류 과제를 만나면 이 순서로

1. **어느 실수가 치명적인가** (오탐/미탐) → 방법 선택: 제재 자동이면 AND(P 1.0), 회수 생명이면 OR, 균형이면 BERT 단독, 무서버·설명 의무면 RULE
2. **누출 금지 3단**: 만들 때 train → 조정할 때 validation → 채점만 test
3. **앙상블은 오답 교집합부터**: AND 오탐=오탐∩ / OR 미탐=미탐∩ — 안 겹칠수록 이득
4. **수치 다음은 병명**: 문맥맹(모델의 몫) / 패턴 구멍(규칙 보완) / 라벨 애매(데이터 검수)
5. **룰 베이스라인은 공짜에 가깝다** — 항상 먼저 만들어 "모델이 이만큼 더 나은가"를 잰다

상세와 실측 근거는 [[export/4_가이드]].
