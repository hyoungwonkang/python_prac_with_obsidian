## 함정 (요약)

- ★ **MLflow 3.x 파일 저장소 유지보수 모드** — ultralytics 뒤에 쓰면 URI가 바뀌어 예외.
  → `sqlite:///mlflow.db`로 명시 고정.
- transformers `classifier MISSING` 경고 = **정상** (새 분류 헤드 초기화 — 그걸 학습시키는 게 파인튜닝).
- 숫자 라벨 데이터면 분류 출력도 숫자 → **새 데이터는 문자열 라벨 권장**.
- YOLO 신규 클래스는 5에폭으론 못 배움 (mAP50 0.06) → 소량 데이터 기준 **40에폭** (0.87).
- ultralytics `val()`은 `project=`를 따로 줘야 잔여 `runs/` 폴더가 안 생김.

git 규칙(datasets-own만 추적)·전체 재현 절차는 [[export/4_가이드]]. 학습 정리는 [[00-학습메모]].
