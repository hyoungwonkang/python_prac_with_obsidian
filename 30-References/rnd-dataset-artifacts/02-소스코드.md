# 02 소스코드 (작업본) — 학습 데이터 산출물 환경

> 보고용 자립본: [[export/2_소스코드]]

## 스크립트 6종 (export/)

- `train_text.py` — **텍스트 분류 범용 학습기**: CSV/디렉터리 → 열·클래스 자동 인식 → 산출물 3종 세트
- `classify_text.py` — 산출물 세트 로드 → 새 텍스트 분류 (재사용 증명)
- `make_yolo_dataset.py` — YOLO 데이터 규약 뼈대 생성 (+`DEMO=1` 합성 데이터)
- `yolo_train_custom.py` — `data.yaml` 규약 → 학습 → 산출물 3종 세트
- `make_pii_schema.py` — ko-pii 33라벨 → `pii_label_schema.md/json` 자동 생성
- `validate_pii_samples.py` — 평가 데이터 라벨을 33라벨 체계에 강제 (종료코드 0/1)

핵심 코드·계약(입력/출력) 설명은 [[export/2_소스코드]]. 함정은 [[04-가이드]].
