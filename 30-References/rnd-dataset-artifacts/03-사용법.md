# 03 사용법 (작업본) — 학습 데이터 산출물 환경

> 보고용 자립본: [[export/3_사용법]]

## 빠른 실행

```bash
cd 30-References/rnd-dataset-artifacts/export

# 새 텍스트 분류기: CSV(text,label)만 준비하면 끝
DATA=/path/to/data.csv NAME=my-clf ~/rnd-env/bin/python train_text.py
ARTIFACT=artifacts/my-clf ~/rnd-env/bin/python classify_text.py "분류할 문장"

# YOLO 커스텀: 뼈대 → 라벨링 → 학습
NAME=my-ds CLASSES=사람,차량 ~/rnd-env/bin/python make_yolo_dataset.py
DATA=datasets-own/my-ds/data.yaml ~/rnd-env/bin/python yolo_train_custom.py

# PII 스키마·검증
~/rnd-env/bin/python make_pii_schema.py && ~/rnd-env/bin/python validate_pii_samples.py
```

파라미터 표·산출물 저장 규칙·MLflow UI는 [[export/3_사용법]].
