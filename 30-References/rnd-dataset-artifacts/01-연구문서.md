## 요지

과제마다 스크립트를 복사·수정하던 방식을 벗어나, **규약만 맞추면 학습→산출물→재사용이
반복되는 환경**을 구성 (상사 지시 목록 1번 — 확장성).

- 순환 구조: 데이터셋(기준의 정본) → 학습 → **산출물 3종 세트**(가중치+label_map+meta) → 새 데이터 분류 → 검수 편입
- 구성 4요소: 텍스트 분류 범용 학습기 / YOLO 커스텀 데이터 규약 / PII 33라벨 스키마+검증 / 산출물 저장 규칙
- 검증(실측): ko스팸 스모크 정확도 0.9367 · YOLO 데모 mAP50 0.8718 · PII 스키마 33라벨(커버 10/33) · 오타 라벨 차단
