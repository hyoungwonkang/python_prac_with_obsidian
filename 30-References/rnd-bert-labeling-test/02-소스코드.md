## 코드 구조 = R&D 3주제 매핑

코드를 위→아래로 읽으면 3주제가 순서대로 드러난다.

| 코드 섹션 | 함수/호출 | R&D 주제 |
|---|---|---|
| ① 데이터·라벨 로드 | `pd.read_csv`, `SpamDataset` | **라벨링 방법론** |
| ② Dataset/DataLoader | `BertTokenizer`, `padding/truncation` | BERT 입력(토크나이즈+패딩) |
| ③ 모델 로드 | `BertForSequenceClassification(num_labels=2)` | **BERT 모델** (패턴 A) |
| ④ 학습 루프 | `AdamW`, `loss.backward()` | BERT 파인튜닝 |
| ⑤ test 평가 | `accuracy_score`·`confusion_matrix`·`classification_report` | **테스트** |

## 핵심 코드

### ③ 모델 — `[CLS]` + 분류층 (BERT 모델)
```python
model = BertForSequenceClassification.from_pretrained(
    "bert-base-uncased", num_labels=2).to(device)   # 109M params
optimizer = AdamW(model.parameters(), lr=2e-5)
```

### ② 입력 준비 — 토크나이즈 + 패딩 (길이 통일)
```python
enc = tokenizer(text, padding="max_length", truncation=True,
                max_length=128, return_tensors="pt")
# input_ids / attention_mask 반환. SMS는 길이 제각각 → 패딩으로 (batch, 128) 통일
```

### ④ 학습 루프 (파인튜닝)
```python
for epoch in range(1, EPOCHS+1):
    model.train()
    for batch in train_loader:
        optimizer.zero_grad()
        out = model(input_ids=..., attention_mask=..., labels=...)
        out.loss.backward()      # 그레이디언트
        optimizer.step()         # 가중치 갱신
```

### ⑤ test 평가 (테스트)
```python
gold, pred = evaluate(model, test_loader)   # 학습 미사용 test셋
print(accuracy_score(gold, pred))
print(confusion_matrix(gold, pred))
print(classification_report(gold, pred, target_names=["ham(0)","spam(1)"]))
```
