# 연구문서 — BERT 모델 · 라벨링 방법론 · 테스트

작성일: 2026-06-26 · 사례: SMS 스팸 분류 · 검증 환경: macOS / Apple Silicon (PyTorch MPS)

## 0. 요약

사전학습된 **BERT** 모델에 분류층을 얹고, **라벨링**된 SMS 데이터로 파인튜닝한 뒤, 학습에 사용하지 않은 **테스트셋**으로 성능을 채점했다. 별도 GPU 없이 로컬 환경에서 3 epoch 학습으로 **테스트 정확도 97.67%**를 달성했다. BERT 모델·라벨링 방법론·테스트 세 주제가 하나의 분류 파이프라인으로 연결됨을 확인했다.

---

## 1. BERT 모델

### 개요
BERT는 문장을 한 단어씩 순차적으로 읽는 대신 **문장 전체를 동시에** 보고, 단어 간 관계(어텐션)를 계산하는 언어 모델이다. 같은 단어라도 문맥에 따라 의미가 달라지는 것을 포착한다.

### 스팸 분류에 적합한 이유
BERT는 문장 맨 앞 `[CLS]` 토큰에 **문장 전체 요약 정보**가 모이도록 사전학습되어 있다. 이 `[CLS]` 벡터 하나에 **분류층**만 추가하면 "문장 → 라벨 하나" 작업이 된다.

```
[CLS] Free entry win prize ... [SEP]
  │
  └─ [CLS] 벡터(768차원) → 분류층 → P(스팸)=0.99 / P(정상)=0.01
```

- 스팸 분류 = "문장 전체에 라벨 하나(스팸/정상)" = 텍스트 분류.
- 구현: `BertForSequenceClassification` (`bert-base-uncased`, 클래스 2개), 파라미터 약 1.1억 개.
- **전이학습(transfer learning)**: 무거운 본체(사전학습 지식)는 재사용하고, 얇은 분류층만 새로 학습한다.

---

## 2. 라벨링 방법론

분류는 **지도학습**이므로 정답(라벨)이 있어야 학습된다. 라벨링과 분류는 동전의 양면 관계다(라벨링=정답 부여 / 분류=그 정답을 맞히기).

### 4단계 절차
| 단계 | 처리 내용 | 핵심 |
|---|---|---|
| 원천 라벨 | SMS Spam Collection 공개 데이터 (사람이 정상/스팸 주석) | 신뢰 가능한 정답 확보 |
| 라벨 인코딩 | 텍스트 라벨 → 숫자 (정상=0, 스팸=1) | 모델 입력 형태로 변환 |
| 클래스 균형화 | 다수 클래스(정상)를 소수 클래스 개수로 언더샘플링 | 편향 방지 |
| 데이터 분할 | 학습 70% / 검증 10% / 테스트 20% | 정직한 평가 기반 |

- 균형화 결과: 원본 정상 4,825건 / 스팸 747건 → 747 / 747 → 분할 후 학습 1,045 / 검증 149 / 테스트 300.
- **원칙**: 결과 품질은 모델보다 **라벨 데이터의 양과 품질**이 좌우한다 (garbage in, garbage out).

## 3. 테스트

### 테스트셋의 정의
학습에 **단 한 번도 사용하지 않고** 별도로 떼어둔 데이터(여기서는 300건, 전체의 20%). 완성된 모델을 처음 보는 데이터로 마지막에 한 번 채점하여 실전(일반화) 성능을 측정한다.

| 세트 | 용도 |
|---|---|
| 학습셋 (1,045) | 가중치 학습 |
| 검증셋 (149) | 학습 중간 점검 / 과적합 확인 |
| 테스트셋 (300) | 최종 성능 채점 (단 1회) |

학습셋으로 채점하면 이미 본 데이터라 성능이 부풀려지고, 검증셋도 학습 중 튜닝에 사용되어 간접 오염된다. 따라서 **독립적인 테스트셋**이라야 정직한 성능 측정이 가능하다.

### 평가 지표
- **정확도(accuracy)**: 맞힌 개수 / 전체.
- **혼동행렬(confusion matrix)**: 오탐(정상→스팸)과 미탐(스팸→정상)을 구분.
- **정밀도/재현율**: 스팸 분류는 **정상 문자를 스팸으로 잘못 차단(오탐)**하는 비용이 크므로 스팸 정밀도가 특히 중요하다.

### 검증 결과 (테스트셋 300건, 학습 미사용)
```
정확도: 0.9767

혼동행렬 [행=실제, 열=예측]        정밀도   재현율   F1
 [[148   1]   ← 실제 정상         정상  0.9610  0.9933  0.9769
  [  6 145]]  ← 실제 스팸         스팸  0.9932  0.9603  0.9764
```
- 정상 149건 중 1건만 스팸으로 오분류 → 정상 문자 오차단이 거의 없음(정밀도 우수).
- 스팸 151건 중 6건 미탐(재현율 0.96).

---

## 4. 결론

사전학습 BERT + 체계적 라벨링(인코딩·균형화·분할) + 독립 테스트셋 채점을 결합하여, 작은 데이터(학습 1,045건)와 짧은 학습(3 epoch), 로컬 환경만으로 **97.67% 정확도**를 달성했다. 세 주제가 단일 파이프라인으로 유기적으로 연결됨을 실증했다.
