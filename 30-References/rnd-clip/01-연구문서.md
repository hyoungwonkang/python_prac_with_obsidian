# 01 연구문서 (작업본) — CLIP 이미지 상황 판단

> 보고용 자립본: [[export/1_연구문서]] · 학습 정리: [[00-학습메모]]

## 요지

상사 지시 4번 — 통합 UXUI 이미지 탭의 "상황 태그" 모듈 선행 R&D. 3단계 완료 (2026-07-09~10).

- **1단계 zero-shot**: 학습 0줄로 실사 상황 판정 (음식 98.1%·동물 99.7%), 프롬프트 교체 = 분류 체계 교체(5초)
- **2단계 linear probe**: 얼린 임베딩 + 머리 1,026눈금(0.9초). 같은 test 27장에서 **잘 쓴 프롬프트의 zero-shot과 동률**(0.7778) — 단 probe는 정밀형(P 0.889), zero-shot은 균형형
- **3단계 KoCLIP**: 영어 모델+한국어 0.5185 붕괴 / **KoCLIP+한국어 0.8519 전체 최고** — `BASE=` 교체만으로 호환
- 발견: 프롬프트 = 하이퍼파라미터 (부정형 0.4444 → 긍정형 0.7778 — CLIP 부정형 약점)
- **UXUI 확정 근거: KoCLIP + 한국어 긍정형 프롬프트** (부정형 금지·대비쌍·받이 클래스)
