# 04 가이드 (작업본) — CLIP을 업무에 쓸 때의 판단 기준

> 보고용 자립본: [[export/4_가이드]]

## 요지

1. **zero-shot / probe / 풀 파인튜닝 선택**: 체계가 자주 바뀌면 zero-shot(프롬프트 교체), 라벨 소량+체계 고정이면 probe(오탐 억제형), 풀 파인튜닝은 대량 라벨일 때만 검토
2. **probe 전에 프롬프트 개선부터 소진** — 문구 수정이 라벨링보다 싸고, 동률까지 가능(실측)
3. **프롬프트 수칙**: 부정형 금지 / 대비쌍 / 받이 클래스 / 언어 일치(한국어=KoCLIP)
4. **UXUI 확정 구성**: YOLO 박스(무엇이 어디) + KoCLIP 상황 태그(전체 맥락) — 상호 보완
5. softmax %는 후보 간 상대 확신일 뿐 — 받이 클래스 + 임계값 운영

실측 근거와 상세는 [[export/4_가이드]].
