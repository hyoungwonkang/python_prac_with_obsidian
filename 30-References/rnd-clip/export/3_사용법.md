# 3. 사용법 — CLIP 이미지 상황 판단 (2026-07-10)

> 실행 환경: `~/rnd-env` (Python 3.12). 모든 명령은 `export/` 폴더 안에서 실행 기준.
> 최초 실행 시 모델 자동 다운로드 (openai CLIP ~600MB, KoCLIP ~600MB).

## 1) zero-shot 상황 판정

```bash
# 데모 (coco128 샘플 8장 × 기본 상황 후보 6개)
~/rnd-env/bin/python clip_zeroshot.py

# 원하는 이미지·상황 후보로 ('|' 구분 — 후보 목록이 곧 분류 체계)
IMAGES=/path/a.jpg,/path/b.jpg \
PROMPTS="a screenshot of a gambling website|a normal daily photo|a document" \
  ~/rnd-env/bin/python clip_zeroshot.py
```

| 파라미터 | 기본값 | 설명 |
|---|---|---|
| `BASE` | openai/clip-vit-base-patch32 | 모델. 한국어는 `Bingsu/clip-vit-base-patch32-ko` |
| `PROMPTS` | 상황 후보 6개 | `\|` 구분. **부정형 금지·대비쌍·캐치올 클래스 포함** |
| `IMAGES` / `N` | coco128 샘플 / 8 | 이미지 지정 또는 샘플 수 |

## 2) 한국어 프롬프트 (KoCLIP — 권장)

```bash
BASE=Bingsu/clip-vit-base-patch32-ko \
PROMPTS="도박 사이트 화면|일상 사진|음식 사진|서류나 문서" \
  ~/rnd-env/bin/python clip_zeroshot.py
```

## 3) linear probe 비교 실험 재현 (사람판별)

```bash
~/rnd-env/bin/python clip_linear_probe.py
# → zero-shot vs probe를 같은 test 27장으로 채점 + 산출물 3종 세트 저장

# 프롬프트 문구 실험 (부정형 vs 긍정형)
ZS_PROMPTS="a photo of animals, food, vehicles or objects|a photo of one or more people" \
  ~/rnd-env/bin/python clip_linear_probe.py
```

| 파라미터 | 기본값 | 설명 |
|---|---|---|
| `EPOCHS` / `LR` | 200 / 0.01 | 선형 머리 학습 (전체 89장 = 1배치) |
| `ZS_PROMPTS` | 부정형 쌍 | zero-shot 대조군 프롬프트 (`\|` 구분, 클래스 순서와 일치) |

## 4) 실험 기록 (MLflow — 2층 관례)

```bash
# 과정 기록(기본): 이 폴더 mlflow.db / 최종 지표: 통합 서버
MLFLOW_URI=http://127.0.0.1:5000 ~/rnd-env/bin/python clip_linear_probe.py

# 로컬 기록 보기
~/rnd-env/bin/mlflow ui --backend-store-uri sqlite:///mlflow.db --port 5001
# 실험: CLIP-제로샷 / CLIP-사람판별-비교
```
