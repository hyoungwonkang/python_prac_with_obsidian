> R&D 산출물 3/4. 자립본: [[export/3_사용법]]. 환경 정본: [[../pytorch-env-hybrid]].

## 0. 환경 (1회) — Python 3.12 필수
기존 `~/ml-env`(3.9.6)는 ko-pii(3.10+)·ultralytics 부적합 → 새 venv.
```bash
brew install python@3.12
/opt/homebrew/opt/python@3.12/bin/python3.12 -m venv ~/rnd-env
source ~/rnd-env/bin/activate
pip install torch transformers datasets seqeval accelerate ultralytics ko-pii mlflow
```

## 실행 (export 폴더에서)
```bash
# NER
NER_SUBSET=6000 EPOCHS=3 ~/rnd-env/bin/python finetune_ner.py   # → F1 + ner_klue.pt
~/rnd-env/bin/python predict_ner.py                             # 개체 추출 데모
# YOLO
EPOCHS=1 ~/rnd-env/bin/python yolo_train.py                     # → mAP + yolo_best.pt
~/rnd-env/bin/python yolo_predict.py                            # 이미지 탐지
# PII (학습 불필요)
~/rnd-env/bin/python pii_detect.py                              # 마스킹 데모
~/rnd-env/bin/python pii_eval.py                                # 커버리지
```

## 결과 해석
- NER `entity F1` (seqeval, 엔티티 단위) · YOLO `mAP50` (객체탐지 표준) · PII `커버리지`(기대 유형 탐지율).

## 문제 해결 (요약)
- `No module ko_pii/ultralytics` → 3.12 venv에서 설치 확인 · KLUE 로드 실패 → `klue/klue` 형식 사용
- 느림 → EPOCHS/SUBSET 축소, **Colab는 자동 전환 말고 협의** · 상세 [[export/3_사용법]].
