# 사용법 — 실행 방법 (How to Run)

macOS Apple Silicon(M4 Max, MPS) 기준 실측 검증. GPU(Colab/CUDA)에서도 코드 변경 없이 동일 실행.

## 0. 환경 설치 (1회) — Python 3.12 필수

기존 `~/ml-env`(3.9.6)는 ko-pii(3.10+)·ultralytics에 부적합 → 3.12 전용 venv를 만든다.
```bash
brew install python@3.12
/opt/homebrew/opt/python@3.12/bin/python3.12 -m venv ~/rnd-env
source ~/rnd-env/bin/activate
pip install torch transformers datasets seqeval accelerate ultralytics ko-pii mlflow
```
검증: `python -c "import torch;print(torch.__version__, torch.backends.mps.is_available())"` → `2.12.1 True`.
(`mlflow`는 `eval_mlflow_ner.py` 실험 기록용 — 그 스크립트를 쓸 때만 필요.)

이하 명령은 `~/rnd-env/bin/python` 으로 실행(또는 venv 활성화 후 `python`). 이 폴더에서 실행.

## 1. NER (개체명 인식)
```bash
# (선택) 데이터 통계 확인 — KLUE-NER 자동 다운로드
~/rnd-env/bin/python ner_dataset.py

# 학습 + 평가 (미니 스모크). 첫 실행 시 klue/bert-base(약 440MB) 자동 다운로드
NER_SUBSET=6000 EPOCHS=3 ~/rnd-env/bin/python finetune_ner.py
#  → entity F1 출력 + ner_klue.pt 저장

# 문장 개체 추출 데모 + 대화형
~/rnd-env/bin/python predict_ner.py

# (선택) 평가 결과 MLflow 기록 — 사전에 MLflow 서버(127.0.0.1:5000) 필요
~/rnd-env/bin/python eval_mlflow_ner.py
```
- 전체 학습은 `NER_SUBSET` 생략(약 21,000건). 파라미터: `EPOCHS`·`BATCH_SIZE`·`MAX_LENGTH`·`LR`.

## 2. YOLO (객체 탐지)
```bash
# 학습(전이) + mAP. 첫 실행 시 yolov8n.pt·COCO128 자동 다운로드
EPOCHS=1 ~/rnd-env/bin/python yolo_train.py     # → mAP50/mAP50-95 + yolo_best.pt

# 이미지 객체 탐지 데모 (COCO128 샘플 자동 선택, 주석 이미지 저장)
~/rnd-env/bin/python yolo_predict.py
SOURCE=/path/to/image.jpg ~/rnd-env/bin/python yolo_predict.py   # 직접 지정
```
- 결과·주석 이미지는 `yolo_runs/` 아래 저장. 데이터는 `<repo>/datasets/coco128/`에 캐시.

## 3. PII (개인정보 탐지·비식별화)
```bash
# 탐지·마스킹 데모 (학습 불필요 — 즉시 실행)
~/rnd-env/bin/python pii_detect.py

# 커버리지 측정 (pii_samples.txt 기준)
~/rnd-env/bin/python pii_eval.py
```

## 5. 문제 해결
| 증상 | 조치 |
|---|---|
| `No module named ko_pii` / `ultralytics` | 3.12 venv에서 설치했는지 확인(§0). 3.9 env엔 ko-pii 설치 불가 |
| KLUE 로드 실패(`Invalid HF URI 'klue'`) | `datasets 5.x`는 `load_dataset("klue/klue","ner")` 형식 필요(코드 반영됨) |
| Apple Silicon 연산 오류 | `PYTORCH_ENABLE_MPS_FALLBACK=1` (코드에 기본 설정됨) |
| 실행이 느림 | `EPOCHS`·`NER_SUBSET` 축소. 무거우면 Colab 검토(**자동 전환하지 말고 먼저 협의**) |
| YOLO 가중치/데이터 다운로드 실패 | 인터넷 연결 확인. 첫 실행만 다운로드, 이후 캐시 사용 |
