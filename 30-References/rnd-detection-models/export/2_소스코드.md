## 1) NER — 핵심 코드

**subword 라벨 정렬(-100)** — `ner_dataset.py`:
```python
def align_labels(examples, tokenizer, max_length):
    tokenized = tokenizer(examples["tokens"], is_split_into_words=True,
                          truncation=True, max_length=max_length, padding="max_length")
    all_labels = []
    for i, labels in enumerate(examples["ner_tags"]):
        word_ids = tokenized.word_ids(batch_index=i)
        prev, aligned = None, []
        for wid in word_ids:
            if wid is None:        aligned.append(-100)       # 특수토큰/패딩
            elif wid != prev:      aligned.append(labels[wid])# 첫 subword만 라벨
            else:                  aligned.append(-100)       # 이어지는 subword
            prev = wid
        all_labels.append(aligned)
    tokenized["labels"] = all_labels
    return tokenized
```

from ultralytics import YOLO
model = YOLO("yolov8n.pt")                       # 사전학습 가중치(자동 다운로드)
