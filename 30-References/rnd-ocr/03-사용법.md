# 03 사용법 (작업본) — OCR 추출 계층

> 보고용 자립본: [[export/3_사용법]]

## 빠른 실행

```bash
cd 30-References/rnd-ocr/export

~/rnd-env/bin/python make_test_images.py                  # ① 테스트 이미지 생성
~/rnd-env/bin/python ocr_eval.py                          # ② EasyOCR 기준선 (CER)
~/rnd-env/bin/python preprocess.py                        # ③ OpenCV 전처리 복구
IMAGES=images_pre ~/rnd-env/bin/python ocr_eval.py        # ④ 복구본 재채점
ENGINE=paddle ~/ocr-env/bin/python ocr_eval.py            # ⑤ PaddleOCR 비교 (격리 환경!)
~/rnd-env/bin/python ocr_pipeline.py                      # ⑥ ②→③ 파이프라인
```

수칙: **ENGINE과 파이썬을 짝 맞출 것** (easy=rnd-env / paddle=ocr-env).
환경 구축(ocr-env 격리 이유 포함)·파라미터는 [[export/3_사용법]].
