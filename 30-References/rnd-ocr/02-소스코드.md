# 02 소스코드 (작업본) — OCR 추출 계층

> 보고용 자립본: [[export/2_소스코드]]

## 파일 5개

- `export/make_test_images.py` — 조건 통제 이미지 생성 (정답 아는 문장 렌더링 + 열화 4종)
- `export/ocr_inspect.py` — 인스펙터(inspector) — 박스·텍스트·신뢰도 원자료
- `export/ocr_eval.py` — 잣대 (엔진 중립 CER — 편집거리·오독 구간)
- `export/preprocess.py` — OpenCV 복구 4종 (deskew·denoise·binarize·upscale)
- `export/ocr_pipeline.py` — ②→③ 미니 파이프라인 (하류 3종 기존 산출물 재사용)

핵심 설계: 인스펙터/잣대 분리(신뢰도는 채점 근거 아님), `make_reader()` 엔진 래핑(실행당
단일 엔진 — torch↔paddle 동거 회피), 전처리 검증은 잣대 재사용(`IMAGES=images_pre`),
파이프라인은 엔진별 환경 서브프로세스+파일 인계(계층 분리와 동형). 상세는 [[export/2_소스코드]].
